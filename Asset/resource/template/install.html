Edit Language
Please note:

The default language of the platform is English. Please note that the system includes by default the dictionaries of the languages selected for the platform and the administration system. You must decide if you want to support multiple languages at this time. Also consider that you should implement the template and dictionary system to expand the capabilities of using multiple languages, as well as including new languages.

The default language of the platform will be used as a last resort, the priority in Language detection is as follows:
1. Language set by end user as default.
2. language detected on the end user's operating system.
3. language set by default on the platform.
